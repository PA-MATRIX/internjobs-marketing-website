Create a new file. This is a TypeScript port of apps/app/src/memory/graph.mjs adapted for Parrot's Worker context. Key differences:
- Receives `env: Pick<Env, "FALKORDB_URL" | "FALKORDB_PASSWORD">` instead of reading process.env (Workers don't have process.env; env is passed at request time).
- Uses Parrot label namespace: `:Employee` / `:Todo` / `:Person` / `:Email` / `:ChatMsg`. NEVER touches `:Student` / `:Fact` / `:Role`.
- Fact semantics: a `:Todo` node is the subject; `:HAS_TODO`, `:MENTIONS` (to `:Person`), `:FROM_EMAIL` / `:FROM_CHAT` (to `:Email` / `:ChatMsg`) are the edges.
- Same graph name: `"internjobs"` (same physical FalkorDB instance, isolation is by label).

**Module structure** (follow graph.mjs section headings as a template):

recordTodoFact(env, {employeeId, sourceChannel, sourceId, title, preview?, urgencyScore, deadlineAt?, mentionedActors?, isMention})

**Fail-soft rule (MANDATORY on every export):**
- If `!env.FALKORDB_URL` → return null / [] / "" immediately, no log spam.
- All graph calls wrapped in try/catch → return null / [] / "" on failure.

**todoHash function:**
```ts
import { createHash } from "node:crypto";
// Workers have nodejs_compat so node:crypto is available.
function todoHash(employeeId: string, sourceId: string): string {
  return createHash("sha256")
    .update(`${employeeId}|${sourceId}`)
    .digest("hex")
    .slice(0, 32);
}
```

**Idempotent duplicate guard for recordTodoFact:**
Use MERGE on the deterministic `id: todoHash(employeeId, sourceId)` — FalkorDB MERGE on an existing node skips the ON CREATE block. This is the "no duplicate insert" guarantee (ROADMAP SC-4). No need for a separate close-out step; the hash deduplication IS the close-out.

**Worker singleton pattern:**
Unlike graph.mjs (Node process singleton), Workers can be evicted. Use a module-level Map keyed by FALKORDB_URL to cache client instances across requests in the same isolate lifetime:

```ts
const _clients = new Map<string, FalkorDB>();
async function getGraphClient(env: Pick<Env, "FALKORDB_URL" | "FALKORDB_PASSWORD">): Promise<FalkorDB | null> {
  const url = env.FALKORDB_URL;
  if (!url) return null;
  if (_clients.has(url)) return _clients.get(url)!;
  // ... connect, store, return
}
```
