Wire the Phase 14 graph helper into the existing Phase 12 AI extraction pipeline:

Purpose: Closes the pre-extraction context injection loop (ROADMAP SC-2) and the post-extraction graph write loop (ROADMAP SC-3). The dashboard's active todo list now reflects graph-level resolution without a cron.

Output: Modified ai.ts (context block parameter) + modified durableObject/index.ts (graph wiring).
