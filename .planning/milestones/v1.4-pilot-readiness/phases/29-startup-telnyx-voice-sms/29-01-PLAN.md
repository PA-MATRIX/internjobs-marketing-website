    **routes/telnyx.ts** — `telnyxRouter` Hono sub-router, export for mounting in app.ts:
    
    ```
    POST /webhooks/telnyx/sms
    ```
    
    Logic (in strict order — do NOT reorder):
    
    1. **STOP handling** (MUST be first, before signature verify):
       Parse body. If `text.trim().toUpperCase()` matches `STOP|STOP\s*ALL|UNSUBSCRIBE|CANCEL|END|QUIT`:
       - Call `resolveChannelLink(env, 'telnyx-sms', fromPhone)` to find the row (may be null for unknown numbers — that is fine, still comply with opt-out per TCPA).
       - If found: `PATCH /v1/channel-links/${rowId}/opt-out` on Fly proxy (set `status='opted_out'`, `opt_in_flags='{}'`). Add this endpoint to index.mjs in Task 2 if you did not yet — add it here if missed.
       - `sendSms(env, fromPhone, "You've been unsubscribed. Reply START to re-subscribe.")`.
       - Return 200 immediately.
    
    2. **Ed25519 signature verification**:
       - Read headers `telnyx-signature-ed25519` and `telnyx-timestamp`.
       - Signed string: `${timestamp}|${rawBody}` (raw body bytes before JSON.parse).
       - Verify using `crypto.subtle.verify('Ed25519', publicKey, signature, signed)` where
         `publicKey` is imported from `env.TELNYX_WEBHOOK_PUBLIC_KEY` (base64-encoded raw key,
         decode with `crypto.subtle.importKey('raw', base64Decode(env.TELNYX_WEBHOOK_PUBLIC_KEY), 'Ed25519', false, ['verify'])`).
       - If `!env.TELNYX_WEBHOOK_PUBLIC_KEY`: skip verification and log warning (ops-deferred mode).
       - If verification fails: return 401 `{error: 'invalid_signature'}`.
    
    3. **Parse event**: extract `fromPhone = payload.data?.payload?.from?.phone_number` and
       `body = payload.data?.payload?.text ?? ''`. Return 200 immediately if
       `payload.data?.event_type !== 'message.received'` (handle delivery receipts silently).
    
    4. **Identity resolution**: `resolveChannelLink(env, 'telnyx-sms', fromPhone)`.
       If null: `sendSms(env, fromPhone, "Hi! To connect your startup to InternJobs, call us at ${env.TELNYX_FROM_NUMBER ?? 'our number'} — we'll get you set up in 30 seconds. Or text INVITE for an onboarding link.")`. Return 200.
    
    5. **Intent classification**: `classifyIntent(body, env)`.
       If null: `sendSms(env, fromPhone, "Didn't catch that. Try: 'show me the top 3 candidates' or 'post a frontend intern role'.")`. Return 200.
    
    6. **Dispatch**:
       - If `intent.kind === 'search'`: call `handleSearch` from `../tools/search` (verify import path matches existing search.ts).
       - Otherwise: call `handleExecute` from `../tools/execute`.
       Pass `{ startup_id: ctx.startup_id, member_id: ctx.member_id, env, action: intent.action, params: intent.args }`.
    
    7. **Format + reply**: `sendSms(env, fromPhone, formatForSms(result))`. Return 200.
    
    Handle all thrown errors with a try/catch: on error, log + `sendSms(env, fromPhone, "Something went wrong. We've been notified.")` if SMS creds are available.
