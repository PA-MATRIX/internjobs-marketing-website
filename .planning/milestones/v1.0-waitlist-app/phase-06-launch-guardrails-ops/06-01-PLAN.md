# 06-01 Plan: Deploy and Smoke Tests

## Goal

Make production deploys safe for both marketing and app.

## Must Haves

- Cloudflare Pages deploy command verifies marketing assets.
- Fly deploy docs include secrets, domain, health check, and rollback notes.
- Fly app belongs to `projecta-labs`, not Growthpods/SIOS.
- Playwright smoke checks cover marketing `/`, `/startups`, `/privacy`, `/terms`.
- App health check is part of verification.

## Verification

- `npm run verify` passes.
- Playwright screenshots show rendered content on desktop and mobile.
