# 06-02 Plan: Privacy, Security, and Operations

## Goal

Ensure production waitlist data collection is controlled and supportable.

## Must Haves

- Webhook endpoints authenticate requests.
- Sensitive data is excluded from logs.
- Privacy deletion/export process is documented.
- Provider credentials live in Infisical only and are synced into runtime providers as needed.
- Failure states are visible enough to debug without exposing user data.

## Verification

- Invalid webhook auth is rejected.
- Logs from test flows do not include sensitive profile/message content.
- Privacy operations checklist exists.
