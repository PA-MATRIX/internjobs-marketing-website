# 05-02 Plan: Student Profile Review

## Goal

Let students review and correct the context InternJobs.ai will use.

## Must Haves

- Student can see known profile basics.
- Student can add interests, projects, and preferred work.
- Student can correct or remove stored context.
- Changes write to Neon with audit events.

## Verification

- Updated profile context persists after refresh.
- Empty optional fields are handled gracefully.
