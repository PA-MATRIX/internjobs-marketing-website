# 02-03 Plan: Student Waitlist Entry

## Goal

Create the first post-auth waitlist screen.

## Must Haves

- Student sees a simple welcome screen after LinkedIn sign-in.
- App creates or updates the student waitlist record by Clerk user ID.
- Student is routed to the channel pairing step.
- The UI stays messaging-first and avoids a long form.

## Verification

- Repeated sign-in updates the same waitlist profile.
- Student can refresh the onboarding page without losing state.
