# 04-02 Plan: QR and Pairing Code UI

## Goal

Show students an easy way to connect their preferred messaging channel.

## Must Haves

- App generates a unique pairing code.
- QR code opens a message to the InternJobs.ai number where supported.
- Screen shows the code as a fallback.
- Pairing codes expire and can be regenerated.

## Verification

- QR/code screen renders on mobile and desktop.
- Expired codes cannot confirm a channel.
